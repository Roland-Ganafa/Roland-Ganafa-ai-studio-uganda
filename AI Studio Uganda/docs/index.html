Sovereign AI for Developers

Integrate Crane models into your applications using simple, secure APIs. Designed for low latency, local language intelligence, and sovereign deployment.

Getting Started

  1. Create an account on the platform
  2. Generate an API key in your dashboard
  3. Choose a Crane model (e.g., crane-v1.0-instruct)
  4. Copy the integration code below

Authentication

All requests use Bearer token authentication.

HTTP Header
Authorization: Bearer YOUR_CRANE_API_KEY

cURL

curl https://api.aistudio.ug/v1/crane/generate \
  -H "Authorization: Bearer $CRANE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{

  }'

JavaScript

const res = await fetch("https://api.aistudio.ug/v1/crane/generate", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.CRANE_API_KEY}`,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "crane-v1.0",
    input: "Summarize this text"
  })
});

const data = await res.json();
console.log(data.output);

Python

import requests

url = "https://api.aistudio.ug/v1/crane/generate"
headers = {
  "Authorization": f"Bearer {API_KEY}",
  "Content-Type": "application/json"
}

payload = {
  "model": "crane-v1.0",
  "input": "Hello world in Runyankole"
}

response = requests.post(url, json=payload, headers=headers)
print(response.json()["output"])

Translation Endpoint

POST /v1/crane/translate

Optimized for East African languages. Required fields:

  • source_lang (e.g., "en")
  • target_lang (e.g., "lg", "sw")
  • text

Embeddings

POST /v1/crane/embed

Use embeddings for semantic search, clustering, and retrieval augmented generation (RAG).

Production Guidance

Timeouts: We recommend a timeout of 30s for standard generation and 60s for long-context tasks.

Retries: Implement exponential backoff for 5xx errors.
